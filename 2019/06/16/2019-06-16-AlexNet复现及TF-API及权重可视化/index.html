TF_APIs_Used_In_Code

  1. tf.nn.conv2d: 这个是首先去查找的Api函数(!!), 它定义在 tensorflow/python/ops/nn_ops.py中, 它computes a 2-D convolution given 4-D input and filter tensors(这里的2-D和4-D别搞混了, 一个是卷积, 一个是Tensor的尺寸), 它的函数形式:

  2. Input和Filter的shape如下:

    Input Tensor Shape Filter/Kernel Tensor Shape
    [batch, in_height, in_width, in_channels] [filter_height, filter_width, in_channels, out_channels]
    输入: [批的大小(数据数), 高, 宽, 输入通道数(如RGB)] 滤核: [高, 宽, 通道数(要被卷积的输入), 通道数(输出的特征图数)]

    下面再讨论下它的参数:

    参数名 参数说明(针对理解和记忆)
    input 变量类型: Tensor(4D), 另: 这里4D中维度order的翻译, 见下面的data_format参数的规定
    filter=None 单元格
    strides=None 单元格
    padding=None 单元格
    use_cudnn_on_gpu=True 单元格
    data_format=’NHWC’ 单元格
    dilations=[1,1,1,1] 单元格
    name=None 单元格
    filters=None 单元格

返回: A Tensor. Has the same type as input, 注: 即返回Tensor的type同输入的type.
